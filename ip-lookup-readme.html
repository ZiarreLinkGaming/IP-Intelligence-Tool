üîó Dual API Integration

üîé Intelligent Input Handling

  • Accepts multiple IPs via:
    • Spaces, commas ,, semicolons ;, or slashes /
  • Supports both standard and defanged IPv4 input:
    • Standard IPv4 (e.g., 8.8.8.8)
    • Defanged IPv4 (e.g., 1[.]1[.]1[.]1) which is automatically converted to 1.1.1.1 for lookup.
  • Robust IPv6 parsing, including:
    • Full (e.g., 2001:0db8:85a3:0000:0000:8a2e:0370:7334)
    • Compressed (e.g., 2001:db8::8a2e:370:7334)
    • IPv6 with zone indices (e.g., fe80::1ff:fe23:4567:890a%eth0 - zone index is automatically removed)
    • IPv4-mapped IPv6 addresses (e.g., ::ffff:192.0.2.1)
  • Automatic cleanup of invalid entries:
    • Non-IP text is automatically ignored or removed.
    • Only valid IPv4/IPv6 addresses are processed.
  • Duplicate IP handling for efficient API usage:
    • Ensures that each unique IP address is queried only once, even if entered multiple times.
    • Prevents unnecessary API calls, conserving your daily lookup limits.
  • Supports Ctrl+A to select all
  • Pressing Enter triggers lookup

‚öôÔ∏è Usage

To get started, follow these steps:

  1. Ensure VPN.ps1 is in the same directory where you plan to create the launcher script.

  2. Create the Launcher Script (run-vpn.cmd):

    To ensure proper execution and minimize potential SmartScreen warnings, it is recommended to manually create this launcher script on your machine. This prevents the file from being flagged by Windows as originating from an untrusted internet download.

    1. Open Notepad or any plain text editor.
    2. Copy the entire code block below and paste it into the new file: 
      @echo off
setlocal

set "scriptDir=%~dp0"

powershell -NoProfile -ExecutionPolicy Bypass -WindowStyle Hidden -File "%scriptDir%VPN.ps1"

endlocal
    3. Save the new file in the same folder as VPN.ps1.
      • Set "Save as type:" to: All Files
      • Enter "File name:" exactly as: run-vpn.cmd

  3. Double-click run-vpn.cmd to launch the tool.

  4. Enter one or more IP addresses in the input box.

    • Accepts separators: space, comma ,, semicolon ;, or slash /
    • Defanged IPs will be refanged automatically.

  5. Click Lookup or press Enter.

  6. View results in the output panel with color-coded flags and full details.

  7. Use the AbuseIPDB link to view extended data in your browser.

  8. Click the "Screenshot" button to capture and save the current output.

üõ†Ô∏è Troubleshooting

If you encounter a "Windows protected your PC" SmartScreen warning when running run-vpn.cmd or VPN.ps1, it means Windows is prompting you about an unrecognized application. This is a standard security feature and the script is safe to run.

This warning typically occurs when files are downloaded from the internet and are not digitally signed. By having you manually create the run-vpn.cmd file (as instructed in the Usage section), the file is treated as if you created it locally, which often helps in preventing or mitigating these warnings.

Should the warning still appear, follow these steps:

  1. In the SmartScreen dialog, click on "More info".
  2. Then, click the "Run anyway" button that appears.

If, for any reason, the .cmd script itself seems to fail or behave unexpectedly after manual creation, ensure you followed the "Create the Launcher Script" steps precisely, especially regarding the file name and "Save as type" setting.
